Gutenberg editor block to display progress of work on a 3D model exported as html file from e-drawings with a chat module to communicate with clients.

== Description ==
Thanks to this block you will be able to upload your HTML file from the e-drawings viewer, show your progress graphically in your post or page and discuss your work with your clients via the chat section.

== Installation ==
Type 3D Work In Progress in plugins search and install. Don\'t forget to visit the Settings Section.

== Screenshots ==
1. Style your favourite gauge.
2. Insert 3D Work In Progress block in your post or page.
3. Show your client your work progress.
4. Chat and take snapshots of the viewed model.
